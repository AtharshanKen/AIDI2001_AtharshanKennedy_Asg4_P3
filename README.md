# Simple To-Do Application

This is a simple To-Do application built with Streamlit and Python. It helps users manage their daily tasks with features like adding, removing, and viewing tasks.

## Features

- Add new tasks with a title and description.
- View a list of all tasks.
- Delete tasks.

## How to Run the Application

To run this Streamlit application, follow these steps:

1. Clone Repo with 'git clone URL' in a directory
    git clone https://github.com/AtharshanKen/AIDI2001_AtharshanKennedy_Asg4_P3.git

2.  Change into that directory in the terminal to where `app.py` is located.
    cd .\AIDI2001_AtharshanKennedy_asg4_P3
    
3.  Run the Streamlit application with 'python -m streamlit run app.py' in the terminal.

    This will open the application in your default web browser (`http://localhost:8501`).

## Project Structure

-   `app.py`: The main Streamlit application file containing the UI and integration logic.
-   `test_app.py`: Unit tests for the `app.py` module, ensuring robust functionality.
-   `prd_for_todo_app.md`: Product Requirements Document detailing the application's features and design decisions.
